# Mi Avatar

Creador de avatar estilo Duolingo para la familia CodersPlug.

**Live:** https://codersplug.github.io/CrackDelBarrio/

(Repo folder still `CrackDelBarrio`; display name is **Mi Avatar**.)

## Qué hace

- Diseñá un avatar: piel, pelo, color, ojos, boca, remera, accesorio.
- Auto-guarda en `localStorage` key `codersplug_kid_avatar` (mismo origin que el resto de los juegos).
- Los juegos de ajedrez muestran esa cara en la pieza jugable (`kid-avatar.js`).

## Archivos

- `kid-avatar.js` — schema + `drawHead` / `drawFull` (copiar idéntico a los otros repos)
- `game.js` — UI del creador
